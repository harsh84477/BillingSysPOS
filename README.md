# Billing Management System (POS)

A professional Point of Sale and Billing Management system built with modern technologies.

## Features

- **Professional Billing**: Generate clean, professional invoices for your customers.
- **Dynamic QR Codes**: Automatic UPI payment QR code generation with pre-filled amounts.
- **Invoice Customization**: Toggle GST numbers, item price breakdowns, and unit price columns from settings.
- **Billing History**: View and track all past transactions with detailed item breakdowns.
- **Multi-Tenant Support**: Isolated data management for different business profiles.
- **Modern UI**: Built with React, Tailwind CSS, and shadcn/ui for a premium look and feel.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Set up your environment variables (Supabase URL and Key).
4. Run the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Database**: Supabase
- **State Management**: TanStack Query (React Query)
- **Icons**: Lucide React
